This plugin allows you to add your portfolio as image gallery.

== Description ==

This plugin makes it possible to add portfolio as a simple image gallery. You can add your portfolio as
a custom post. In every post you can add your portfolio title, image, live project url and description
of the portfolio within 600 characters. When you click on any of your portfolio, you can see details
of your portfolio with a pop-up modal.

= Features =

* Fully responsive.
* Easy to integrate new portfolio.
* Add live project url.
* Upload any number of portfolio
* Add gallery using shortcode to a page

== Installation ==

1. Upload the `portfolio-image-gallery` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.
3. Add shortcode `[portfolio-image-gallery]` to your page.
4. Add portfolio to your Admin `Portfolios` Menu page.

== Frequently Asked Questions ==

= How to use the plugin? =

1. Click 'Add New' in the 'Portfolios' menu and fill out your page.
2. Upload pictures via the uploader at the 'Featured Image' option. 
3. Add title of the Portfolio.
4. Add content of the Portfolio within 600 characters.
5. Add live project url in 'Portfolio options' of metabox.
